## 4.33.0 训练锁定的值不是永远不变的

Write Leveling（[DDR5-S4.21-写均衡训练]）确定了 Host DQS 的相对 CK 的最优偏移——tDQSoffset。Read Leveling 确定了 DQS Gate 的最佳开启时刻——tDQSCK。但这两个值都是在**训练那一刻**的温度和电压条件下的最优解。当系统从冷启动升温到 85°C 满载运行——CMOS 门的延迟会发生变化。

**tRX_DQS2DQ** 是 DRAM 内部从 DQS 采样沿到 DQ 数据稳定之间的延迟。它在 25°C→85°C 的跨度内可能漂移 12~30 ps。DDR5-6400 下 1 UI = 156 ps——30 ps ≈ 0.2 UI，这在高速模式下已经是不可能忽略的裕量损失。

补偿策略分三档：**重训练**（最彻底，每几百 ms 一次）→ **温度曲线补偿**（按 MR4 温度读数 + 预先标定曲线调 MR40，几乎零开销）→ **周期性 ZQ 校准**的同时顺便做 Read DQS 快速检查。当温度跳变超过 ~20°C 时，建议依次重做 VrefDQ → Read Leveling → DFE Training。
